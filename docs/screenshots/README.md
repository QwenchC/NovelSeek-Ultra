|--------|----------|
| `01_home_overview.png` | 应用主界面，底部导航栏全部可见，左张为空状态，右张为有项目卡片的状态 |
| `02_settings_overview.png` | 设置页完整截图，显示语言切换、文本模型、图片引擎、备份等区块 |

| `03_short_outline_generate.png` | 大纲生成页，显示世界观/时间线/额外要求输入框、章节数量滑块，以及 AI 流式输出中的大纲文本 |
| `03_short_outline_save.png` | 保存大纲对话框，显示各字段勾选框（世界观/时间线/章节等）和"确认"按钮 |
